# requirements.txt Generator CLI

## Usage

```bash
$ reqgene <directory path> <language> [option]
```

## Options

    -h, --help:
        Show this help message and exit

    -v, --version:
        Write the version to a file

    -j, --jupyter:
        For Jupyter Notebook

    -d, --detail:
        Display file information
